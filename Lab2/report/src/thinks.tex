\section*{Выводы}

\begin{enumerate}
    \item Программа демонстрирует ускорение в диапазоне от $1.3\times$ до $3\times$.
    \item Многопоточность малоэффективна для малых матриц из-за высоких накладных расходов.
    \item При размерах от $25\times 25$ ускорение превышает $70\%$ эффективности.
    \item Оптимальное количество потоков соответствует числу физических ядер процессора.
    \item Алгоритм корректно работает в многопоточном режиме и демонстрирует преимущества параллелизма.
\end{enumerate}
